# Templates

模板用于建立结构一致的新文档。复制后必须删除不适用占位符，并填入真实状态、边界和证据。

简单的 README、链接或文字修正无需套用模板，直接提交即可。团队参与流程见 [`CONTRIBUTING.md`](../CONTRIBUTING.md)。

## 使用规则

1. 先从 `WORKSPACE.md` 确认归属。
2. 选择最接近的模板，不为轻量文档强行填满所有章节。
3. 保留状态、适用范围、Owner、验证和复审信息。
4. 更新目标目录的 `README.md`。
5. 模板演进不自动修改已创建文档。
