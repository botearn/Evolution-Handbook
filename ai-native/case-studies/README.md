# AI Native Case Studies

案例用于为原则和模式提供内部证据。

每个案例至少记录：

- 背景和目标
- 原有工作方式
- 引入的 AI Native 实践
- 人、Agent 和工具的责任分配
- 验证方式与结果
- 失败、限制和意外影响
- 可以复用的结论

案例不是宣传稿。没有结果和限制的内容不应进入本目录。
