# AI Native Patterns

Pattern 描述在特定条件下可重复采用的结构，同时保留代价和不适用场景。

## 当前内容

- [真源与 Bridge](source-of-truth-and-bridges.md) - `Pattern Candidate`
- [可回看的 Agent 工作台](observable-agent-workspace.md) - `Pattern Candidate`

## Pattern 最小结构

- 问题与触发条件
- 推荐结构
- 适用与不适用场景
- 代价、风险和替代方案
- 至少一个采用证据
- 验证和退出条件

新增 Pattern 可以先从最小结构开始；没有项目证据时保持 `Pattern Candidate`。
