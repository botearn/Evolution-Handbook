# Useful Skills

这里记录团队觉得好用、值得复用的 Skill、Hook、MCP 和 Agent 配置。

## 当前内容

- [Skill 清单](skill-catalog.md)

## 记录什么

适合放这里：

- 常用 Skill
- 好用的 Hook
- MCP server 使用边界
- Agent 配置模板
- 适合复用的 prompt / workflow

不要放这里：

- API key、token、账号信息
- 私有 URL
- 只适合一个人的本地偏好
- 已经属于某个项目仓库的实现细节

## 每条记录至少写清

- 用来解决什么问题
- 什么时候用
- 什么时候不要用
- 需要哪些权限
- 怎么验证结果
