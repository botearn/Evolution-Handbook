# AI Governance

这里记录 AI 特有的组织治理规则，包括：

- 风险分级与自动化级别
- Agent 权限和批准机制
- 敏感数据与模型供应商边界
- 审计日志和结果可追踪性
- 模型、Prompt、Skill 和评估的变更管理

通用安全与组织治理仍以 `governance/` 为真源。本目录只记录 AI 场景的扩展规则。
