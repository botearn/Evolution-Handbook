## 准入条件

一项实践进入本目录前，应：

- 至少适用于两个项目，或明确设计为跨项目默认值
- 有真实采用证据或清晰的验证方法
- 说明不适用场景和例外处理
- 能被转化为检查清单、模板或可观察行为

项目特有做法先留在 `projects/<project>/`，验证后再晋升。
